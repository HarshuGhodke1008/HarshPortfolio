# Portfolio Website

A simple and elegant portfolio website designed to showcase skills, education, and projects. Built with HTML and CSS to ensure responsiveness and a clean user interface.

## 🌟 Features
- **About Me:** A brief introduction and overview of professional goals.
- **Education:** Academic background and achievements.
- **Projects:** Key projects with descriptions and technologies used.
- **Navigation:** Easy access to different sections via the navigation menu.

## 🛠️ Technologies Used
- **HTML5**: For the structure of the website.
- **CSS3**: For styling and layout.

## 📚 Sections Overview
1. **Header:** Contains the name and navigation links.
2. **About:** Brief introduction to the user.
3. **Education:** Details academic milestones and achievements.
4. **Projects:** Highlights key projects with roles and technologies.
5. **Footer:** Contains copyright information.

## 🚀 How to Use
1. Clone or download the project files.
2. Ensure the `index.html` and `styles.css` are in the same directory.
3. Open `index.html` in a browser to view the website.
